# ROADMAP

## 当前阶段：平台产品化

项目定位：面向所有人的静态博客平台，替代"丑/停更/过时"的现有方案。

## 战略优先级

1. **P0 — 构建时 Markdown → HTML 渲染**（SEO 致命缺陷）
2. **P0 — package.json + CLI**（无法分发）
3. **P1 — 开发体验补全**（热重载、草稿系统）
4. **P1 — 生态基础**（RSS、sitemap、i18n、暗色模式、字体加载）
5. **P2 — 功能扩展**（评论、搜索优化、数学公式等）

## 核心约束

- 零依赖（不引入 npm 包）
- 零依赖优先于功能完整
- 安全第一（不暴露路径、不执行用户输入）
- 构建产物干净（dev-only 代码不进入 dist/）
