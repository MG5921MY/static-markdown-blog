# 关于本站

这是一个基于 Docker 部署的静态博客系统演示站点。

## 特性

- 📝 Markdown 文章支持
- 🎨 8 套内置主题
- 📂 灵活的分类系统
- 💭 瞬间/说说功能
- 🔗 友情链接管理
- 🖼️ 图库相册功能
- 🔒 内置安全防护
- 🐳 Docker 一键部署
- ⚡ 纯静态，无需数据库

## 功能模块

| 功能 | 说明 | 入口 |
|------|------|------|
| 瞬间 | 记录生活点滴，时间线展示 | [查看](./moments.html) |
| 友链 | 友情链接，卡片式布局 | [查看](./links.html) |
| 图库 | 图片相册，瀑布流展示 | [查看](./gallery.html) |

## 开始使用

这是默认的演示站点。要创建您自己的博客：

1. 准备配置文件和文章
2. 挂载到容器对应目录
3. 重启容器

详细步骤请查看 [快速开始](./page.html?id=guide) 指南。

## 技术栈

- 前端：原生 HTML/CSS/JavaScript
- 构建：Node.js
- 部署：Docker
- 配置：YAML
