# 安全特性

博客系统内置多项安全防护措施。

## 前端安全

- **XSS 防护** - 所有输入经过转义，Markdown 使用 DOMPurify 净化
- **路径安全** - Hash ID 隐藏真实路径，防止路径遍历攻击
- **URL 验证** - 阻止 javascript:、data: 等危险协议

## Docker 安全

- **非 root 用户** - 容器以普通用户运行
- **只读文件系统** - 减少攻击面
- **资源限制** - 限制 CPU 和内存使用
- **无额外依赖** - 零 npm 包，避免供应链攻击

## 最佳实践

1. 使用 HTTPS 部署
2. 定期更新镜像
3. 不要在配置中存储敏感信息
