# 分类显示模式

博客支持两种文章分类显示模式：平铺模式和层级模式。

## Flat 平铺模式

所有文章以卡片形式直接展示，适合文章数量较少的分类。

```yaml
categories:
  - id: tools
    name: 工具记录
    path: posts/tools
    type: flat  # 平铺模式
```

特点：
- 所有文章一目了然
- 卡片式布局
- 适合扁平目录结构

## Tree 层级模式

按目录层级展示，支持多级子目录导航。

```yaml
categories:
  - id: tech
    name: 技术笔记
    path: posts/tech
    type: tree  # 层级模式
```

特点：
- 支持多级目录
- 目录树导航面板
- 面包屑导航
- 适合内容丰富的分类

## 目录树面板

在层级模式下，点击右侧的 🗂️ 按钮可以打开目录树面板，快速浏览和跳转到任意目录。
